- 전면 처리 -
명령어를 입력하면 명령어가 전면에서 실행되며 명령어 실행이 끝날 때까지 쉘이 기다려 줌

- 후면 처리 -
명령어들을 후면에서 처리하고 전면에서는 다른 작업을 할 수 있으면 동시에 여러 작업을 수행할 수 있다. 
사용법 : $ 명령어 &

- 프로세스 -
실행중인 프로그램을 프로세스(process)라고 부르고, 각 프로세스는 유일한 프로세스 번호 PID를 갖습니다.

& 프로세스 관련 명령어
   & ps : 현재 존재하는 프로세스들의 실행 상태를 요약해서 출력
   & ps <옵션>
          ㄴ-e : 시스템 전체 프로세스 출력
          ㄴ-f : 자세한 정보
          ㄴ-ef : 모든 프로세스 + 상세 정보
          ㄴ-u 사용자 : 특정 사용자 프로세스만 출력
          ㄴaux : 전체 보기
  & sleep <초> : 해당 시간만큼 프로세스 중지
  & kill <프로세스 번호 OR %작업번호> : 해당 프로세스 번호 / 작업번호(프로세스 실행 순서)에 해당하는 프로세스를 강제 종료 시킴
  & wait <프로세스번호> : 해당 프로세스 번호가 종료할떄까지 기다림
  & exit <종료코드> : 쉘을 종료하고 종료코드(exit code)을 부모 프로세스에 전달

- Bash -
Linux/Unix 시스템에서 사용하는 명령어 해석기(shell) 이고, 사용자가 입력한 명령을 해석해서 OS에 전달하는 역할을 합니다.

- 여러 명령어 -
$ alias 이름=문자열 : 문자열의 명령을 <이름>이라는 별칭으로 축약해서 사용 가능함
$ history [-r/h] [번호] : r(read), h(help)이고, 번호를 입력하면 최근에 사용한 명령을 번호값만큼 보여줌 
$ !! : 바로 전 명령 실행
& !n : 이벤트 번호가 인 명령 재실행
! 시작스트링 : 시작스트링으로 시작하는 최근의 명령어를 실행
!? 서브스트링 : 서브스트링을 포함하는 최근의 명령어를 실행
